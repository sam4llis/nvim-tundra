# nvim-tundra

A dark Neovim colorscheme written in Lua. Requires Neovim 0.7.


## Installation

You can install this colorscheme with your favourite package manager. As an
example, for [vim-plug](https://github.com/junegunn/vim-plug):

```
Plug 'sam4llis/nvim-tundra'
```


## Features

More features will be added to Tundra over the next couple of months. Tundra
currently supports [nvim-treesitter](https://github.com/nvim-treesitter/nvim-treesitter)
and italic text by default. Features such as toggling transparent background,
toggling italic text, and toggling bold text are being considered in the Tundra
development road-map.


## Usage

### Getting started

Tundra can be initialised using the following command in an `init.lua` file:

```
require('nvim-tundra').setup()
```
